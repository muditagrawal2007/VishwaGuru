The frontend of VishwaGuru is a React application built with Vite and styled using Tailwind CSS. It provides users with an interface to report civic issues and receive AI-generated action plans.

## Responsibilities

- Collect civic issue details from users
- Send requests to the FastAPI backend
- Display submission status and feedback
- Render AI-generated responses for users

## Tech Stack

- React
- Vite
- Tailwind CSS
