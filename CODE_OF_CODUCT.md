## Code of Conduct
### Our Pledge

***VishwaGuru*** is an open-source civic platform built to empower participation in democracy.
To foster a safe, inclusive, and productive community, we pledge to make participation in this project respectful and harassment-free for everyone.

This Code of Conduct applies to all project spaces, including but not limited to:

+ GitHub repositories (issues, pull requests, discussions)
+ Code reviews and comments
+ Community chats and forums
+ Any public or private communication related to the project

### Our Values

***We value:***

+ Respectful dialogue, even during disagreement
+ Inclusivity across language, region, gender, caste, religion, and political belief
+ Constructive collaboration and learning
+ Accountability and transparency

This project welcomes contributors of all backgrounds and experience levels.

### Expected Behavior

All contributors are expected to:

+ Be respectful and considerate in language and actions
+ Engage in good faith discussions, especially on sensitive civic or political topics
+ Accept constructive feedback gracefully
+ Focus on what is best for the community and the project
+ Use welcoming and inclusive language
+ Assume positive intent, but take responsibility for impact

### Unacceptable Behavior

The following behaviors are not tolerated:

+ Harassment, discrimination, or hate speech of any kind
+ Personal attacks, insults, or derogatory comments
+ Trolling, baiting, or deliberate disruption of discussions
+ Doxxing or sharing private information without consent
+ Threats of violence or intimidation
+ Sexist, casteist, communal, or exclusionary language
+ Political propaganda unrelated to the projectâ€™s goals
+ Any behavior that creates an unsafe or hostile environment
+ This applies regardless of intent.

### Scope

This Code of Conduct applies within all project spaces and also applies when an individual is representing the project or its community in public spaces.

***Enforcement**

Project maintainers are responsible for enforcing this Code of Conduct.

If unacceptable behavior occurs, maintainers may take appropriate action, including:

+ Warning the individual
+ Temporary or permanent removal from project spaces
+ Reverting or rejecting contributions
+ Reporting behavior to relevant platforms if necessary
+ Enforcement decisions will be made in good faith and with the goal of maintaining a healthy community.

### Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it privately to the maintainers.

All reports will be handled confidentially.
We are committed to protecting reporters from retaliation.

### Attribution

This Code of Conduct is inspired by the Contributor Covenant and adapted for the civic, political, and cultural context of VishwaGuru.

###Final Note

VishwaGuru exists to strengthen democratic participation, not to amplify harm or division.
We expect all contributors to uphold that responsibility.
